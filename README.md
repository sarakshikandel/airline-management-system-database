# Airline Management System Database

This repository contains a phase based SQL database project for an airline management system. The system was designed to support passengers, pilots, flights, airlines, route information, database transactions, and operational reporting.

The project includes an enhanced entity relationship diagram, normalized relational schema, SQL database scripts, stored procedures, operator views, and a Python based application interface.

## Project Phases

### Phase 1: Requirements and Schema
Defined the project scope, core entities, relationships, and initial schema design for the airline management system.

### Phase 2: Database Design
Developed the relational database schema, constraints, and SQL structure to support data integrity and organized airline operations.

### Phase 3: Database Implementation
Built SQL mechanics for the database, including table logic, stored procedures, and operational database functionality.

### Phase 4: Application Interface
Created a Python based interface to interact with the airline management system and support user facing database operations.
